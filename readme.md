# HGSS Save Editor

A Zig project using Nix.

## Development

Enter the development shell:

```bash
nix develop
```

Build the project:

```bash
zig build
```

Run the project:

```bash
zig build run
```
Use Flag -Dwindows to build for win systems.
```bash
zig build -Dwindows run
```

## Building with Nix
